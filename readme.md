# Project Break - Dashboard Interactivo

¡Hola! Soy estudiante de **Full Stack Developer** y este es mi primer proyecto integral. Se trata de un **Dashboard Personalizado**, una aplicación web diseñada para centralizar herramientas útiles en una sola interfaz estética y funcional.

## 🚀 Sobre el Proyecto
El objetivo de este Dashboard es poner en práctica conocimientos de **HTML5, CSS3 y JavaScript**, incluyendo el manejo de APIs asíncronas y almacenamiento local.

## 🛠️ Funcionalidades Principales

El dashboard cuenta con cuatro módulos principales:

1.  **Generador de Enlaces (Links)**: 
    * Permite guardar tus URLs favoritas con un nombre personalizado.
    * Utiliza `localStorage` para que tus enlaces no se borren al cerrar el navegador.
2.  **Reloj Digital y Calendario**: 
    * Muestra la hora en tiempo real con un diseño estilo "Casio Vintage".
    * Incluye mensajes dinámicos que cambian según la hora del día.
3.  **Clima (Weather)**: 
    * Conexión con API externa para mostrar el estado del tiempo actual en Donostia.
    * Visualización de temperatura, humedad, viento y pronóstico por horas.
4.  **Generador de Contraseñas Seguras**: 
    * Herramienta para crear claves aleatorias con longitud personalizable.
    * Garantiza el uso de mayúsculas, minúsculas, números y símbolos.

## 🎨 Diseño y Experiencia de Usuario

* **Fondo Dinámico**: El proyecto incluye una cuidada selección de fondos de pantalla que cambian para ofrecer una experiencia visual inmersiva.
* **Index Común**: Todo el proyecto está unificado en una vista principal organizada mediante **CSS Grid**, mostrando todas las herramientas de forma clara y equilibrada.
* **Responsive Design**: El dashboard es totalmente adaptable, reorganizando las columnas en una sola fila para dispositivos móviles (768px).

## 💻 Tecnologías Utilizadas
* **HTML5** - Estructura semántica.
* **CSS3** - Diseño con Grid, Flexbox y variables personalizadas.
* **JavaScript** - Lógica, manipulación del DOM y Fetch API.
